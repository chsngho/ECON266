2. Second, lets plot the data. Using ggplot2 make a scatter plot of the returns of Tesla as the dependent variable and the returns of market as the independent variable. To this, load the ggplot2 package. For the command, here is a rough schema (that you have to fill in yourself):

data %>%
  ggplot(aes (x =ret_sp500, y =ret_tsla) ) +
  geom_point ()

The first part tells R to make a ggplot object, using which dataset, x, and y variables are specified. The second part says to create a scatter plot.

3. Now lets add a best fit line. To do this we copy and paste the same opening as above with an additional command telling R to add the line:

data %>%
  ggplot(aes (x =ret_sp500, y =ret_tsla)) +
  geom_point () +
  geom_smooth (method = 'lm', se = FALSE)
## `geom_smooth()` using formula = 'y ~ x'
